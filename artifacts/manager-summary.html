⚡ Executive Summary

The FPL Team Picker is a comprehensive full-stack web application for Fantasy Premier League optimization. Built on analysis of 5 research sources — including two arXiv papers (2505.02170v1/v2), the FPL_LP_GA GitHub repository, and FPL analytics best practices — it delivers a research-backed, production-grade optimization platform that surpasses every existing open-source FPL tool by combining ML prediction, mathematical optimization, transfer planning, and chip strategy into a single cohesive application with a modern web UI.

📚 Research-Backed Design Decisions

ARIMA(1,0,0) Primary
Best single predictor at 704 cumulative points over 12 gameweeks, confirmed by arXiv 2505.02170v2.
ILP Over Heuristics
Provably optimal solutions via PuLP/CBC. All 5 sources converge on ILP as the gold standard.
No Robust Optimization
Research showed worst-case hedging “sacrificed upside without reducing downside.” Deterministic wins.
3-5-2 Optimal Formation
Emergent optimal across all methods. Midfielders provide best value per million spent.
Hybrid ML λ=2/3
Ridge + XGBoost blend weighted 2/3 toward predictions outperforms realized-point weighting.
ICT & xGI Features
SHAP analysis confirms ICT index and xGI as the most predictive features across all positions.

# Open in browser
$ open http://localhost:3000
